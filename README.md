# 伤害公式对比可视化工具

这是一个用于比较新旧伤害公式差异的可视化工具。通过热力图展示不同攻击力和防御力组合下的伤害差异。

## 功能特点

- 可视化新旧伤害公式的差异
- 可调整的参数：
  - 旧公式攻击力倍率（默认225%）
  - 新公式攻击力倍率（默认78%）
  - 新公式最终倍率（默认4.0）
- 使用热力图展示伤害差异
  - 蓝色区域表示新公式伤害更高
  - 红色区域表示旧公式伤害更高
- 显示最大、最小和平均伤害差异

## 安装和运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
streamlit run app.py
```

## 使用说明

1. 启动应用后，您可以通过滑块调整三个参数：
   - 旧公式攻击力倍率
   - 新公式攻击力倍率
   - 新公式最终倍率

2. 热力图会自动更新，展示参数调整后的伤害差异：
   - 横轴表示攻击力（0-1000）
   - 纵轴表示防御力（0-1000）
   - 颜色表示新公式与旧公式的伤害差异

3. 页面底部会显示一些统计信息，帮助您更好地理解伤害变化。
